# 👩‍💻 Personal Portfolio Website
A professional, responsive personal branding website designed to showcase my journey as a Software Engineering student.

## 🚀 Live Demo
**[Explore my Portfolio here](https://larasaed.github.io/LaraSaed/)**

## ✨ Features
* **Modern UI:** Clean, minimalist design with a focus on readability and professional aesthetics.
* **Navigation:** Fully functional sidebar navigation for quick access to Experience, Education, and Skills.
* **Responsive Design:** Optimized for seamless viewing on different screen sizes.

## 🛠️ Built With
* **HTML5:** For structured and semantic content.
* **CSS3:** For custom layouts, color palettes, and styling.

### 🎓 Academic Context
This project was developed in **2024** as part of the **Web Technologies** course during my **Software Engineering** studies at **Bethlehem University**. It served as a foundational project for mastering UI structure and personal branding.
